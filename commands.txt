setEvent: setEvent    To set a new Event: `setEvent name startTime endTime`
quitCommand: quit     To quit the program
deleteEvent: delete    Delete an event : `deleteCommand eventName eventStartTime`
displayCommand: display     display all the events :  `displayCommand`
helpCommand: help    To get help. This command cannot be changed : `help`
changeCommandShortcut: change     To change a command : `changeCommand commandName newName`
nowCommand: now     To find out what is the present and next scheduled event
